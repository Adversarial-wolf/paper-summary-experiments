# SaTML 2026 Papers - Data Scientist Perspective

## Key Technical Contributions

### AI Fingerprinting & Attribution Security
- **Smudged Fingerprints**: Systematic evaluation of AI image fingerprinting robustness under adversarial conditions
- **LLM Fingerprinting**: Analysis of robustness of language model fingerprinting techniques
- Novel attack/defense methodologies for model attribution security

### Privacy Attacks & Membership Inference
- **Time Series MIA**: Membership inference attacks on forecasting models
- **FedSpy-LLM**: Gradient-based data reconstruction attacks on federated LLMs
- **DP Forests**: Differential privacy effectiveness in Random Forests
- **DeepLeak**: Privacy enhancements for model explanations against leakage

### LLM Security & Safety
- **RobustRAG**: Certifiably robust retrieval-augmented generation against corruption
- **CaMeL**: Prompt injection defense using control/data flow separation
- **Targeting Alignment**: Safety classifier extraction from aligned LLMs
- **BinaryShield**: Privacy-preserving threat intelligence sharing

### Model Robustness & Verification
- **Optimal Robust Recourse**: Lp-bounded model change for optimal robustness
- **Cascading Verification**: Multi-scale robustness verification
- **RobustBlack**: Black-box robustness testing framework
- **TEA Attacks**: Targeted adversarial examples methodology

### Federated Learning Security
- **FedSpy-LLM**: Gradient analysis for federated LLM data reconstruction
- **FL Fragility**: Contribution evaluation distortions in federated learning
- **CPCL with DP**: Constrained private collaborative learning

### Model Unlearning & Privacy
- **Deep Unlearning**: Comprehensive fact removal in language models
- **Exact Unlearning**: SIFT-Masks for one-shot model merging
- **Gauss-Newton Unlearning**: Second-order methods for unlearning
- **Oblivious Unlearning**: Encrypted exact unlearning for ERTs

### Physical & Systems Security
- **Kraken**: EM side-channel attacks on DNNs
- **Temporal Misalignment**: Time-based vulnerabilities in ML systems
- **CHAI**: Embodied AI system hijacking vulnerabilities

### Fairness & Explainability
- **Fair GNNs**: Homophily-aware supervised contrastive fair graph neural networks
- **DeepLeak Explanations**: Privacy in post-hoc explanation methods
- **OOD Detection**: Out-of-distribution detection methods

### Efficiency & Optimization
- **DP-SGD Optimization**: Efficient differentially private deep learning with correct Poisson subsampling
- **StriaNet**: Fast secure inference with homomorphic encryption
- **Wasserstein Perturbations**: Global explainability framework using Wasserstein metrics
- **Efficient SSAT**: Latent clustering for semi-supervised adversarial training

### Enterprise AI Security
- **Adversarial Trading**: Manipulating headlines in LLM-driven algorithmic trading
- **C-Level GenAI**: Enterprise security perspectives
- **Transfer Learning**: Privacy risks in transfer learning scenarios
